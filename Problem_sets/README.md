# Problem Sets

Competition and research-level mathematics problem sets used in arXiv:2602.13695.

## Contents

| Subfolder | Source | Problems | Format |
|-----------|--------|----------|--------|
| `ICCM_problem_sets/test_1/` | ICCM Round 1 (2024) | 6 | `.txt` |
| `ICCM_problem_sets/test_2/` | ICCM Round 2 (2024) | 6 | `.txt` |
| `ICCM_problem_sets/test_3/` | ICCM 2025 | 6 | `.txt` |
| `First_Proof_problem_set/` | First Proof (research-level) | 10 | `.pdf` + `.txt` |

## Naming Convention

ICCM files follow the pattern `test_{round}_{problem}.txt`, e.g. `test_1_3.txt` = Round 1, Problem 3.

First Proof files follow the pattern `test_5_{n}.txt`, e.g. `test_5_7.txt` = Problem 7.
